## Español
 
Plataforma web educativa para alumnos de escuela y liceo con proyectos de **robótica, electrónica, programación y videojuegos educativos**.
 
### 📚 Contenidos
 
La plataforma incluye actividades organizadas por grado escolar:
 
- **Tercero a Sexto (Primaria):** WeDo, Micro:bit, MakeCode Arcade, Scratch
- **Secundaria:** SPIKE Prime, programación avanzada, proyectos autónomos
### 🚀 Características
 
- Sistema de login simple por grado
- Panel de administración para controlar visibilidad de contenidos
- Guías interactivas paso a paso
- Conexión con base de datos Supabase
- Diseño responsive y colorido para primaria, tech oscuro para secundaria
### 🛠️ Tecnologías
 
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Supabase (base de datos)
- Vercel (hosting)
 
## English
 
Educational web platform for elementary and high school students featuring **robotics, electronics, programming, and educational video game projects**.
 
### 📚 Contents
 
The platform includes activities organized by grade level:
 
- **3rd to 6th Grade (Elementary):** WeDo, Micro:bit, MakeCode Arcade, Scratch
- **Middle/High School:** SPIKE Prime, advanced programming, autonomous projects
### 🚀 Features
 
- Simple grade-based login system
- Admin panel to control content visibility
- Interactive step-by-step guides
- Supabase database integration
- Responsive design: colorful for elementary, dark tech theme for high school
### 🛠️ Technologies
 
- HTML5, CSS3, JavaScript
- Bootstrap 5
- Supabase (database)
- Vercel (hosting)
